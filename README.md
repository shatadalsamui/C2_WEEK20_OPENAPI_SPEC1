# OpenAPI Specification Project

This project implements an API based on the OpenAPI specification defined in `openapispec.yaml`.

## Features
- TypeScript implementation
- OpenAPI 3.0 specification
- Node.js runtime

## Setup
1. Install dependencies:
```bash
npm install
```

2. Build TypeScript files:
```bash
npm run build
```

3. Start the server:
```bash
npm start
```

## API Documentation

The API specification is available in `openapispec.yaml`. You can view it using:
- [Swagger Editor](https://editor.swagger.io/)
- [Redoc](https://redocly.github.io/redoc/)

## Development
- Source files: `index.ts`
- Compiled output: `index.js`
- TypeScript config: `tsconfig.json`
